# WoofX3 UI documentation

This site captures **architecture**, **patterns**, and **design decisions** for the woofx3-ui control plane (React + Convex + Vite).

The live product code lives in this repository; the engine itself is separate ([woofx3](https://github.com/wolfymaster/woofx3)).

## Where to start

- **[UI areas](/ui/overview)** — routes, shell, and how each major screen works today
- **[Patterns](/patterns/)** — conventions and where to record design decisions

## Local preview

From the repo root:

```bash
bun run docs
```

Build static output:

```bash
bun run docs:build
```
